Design notes

Color legend: killed. The step cards already carry their own color coding — an engineer reading this picks up the visual grouping without a separate key explaining it. That's explaining something that's already self-evident.

Icons: every SVG maps to what the item actually does. Clipboard for intake. Crosshair for grid-plotting. Send icon for outreach. Magnifying glass with heart for examining feelings beneath wants.

Metric pills and data layer fields use field-name formatting (days_inactive, scorecard_active) so an engineer immediately sees the data model, not a summary of it.
